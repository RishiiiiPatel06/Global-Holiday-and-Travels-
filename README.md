# Global Holiday & Travels

A travel and holiday planning web application that enables users to explore worldwide public holidays, discover destinations, and build itineraries that can be exported and shared. The project aims to merge public-holiday data with practical travel planning rather than functioning as another static tourism platform.

---

## Overview
Global Holiday & Travels allows users to:
- Search public holidays by country and year.
- Discover travel destinations linked to specific holiday periods.
- Create and save day-wise itineraries.
- Export itineraries in PDF or JSON format.
- Maintain responsive accessibility on both desktop and mobile.
